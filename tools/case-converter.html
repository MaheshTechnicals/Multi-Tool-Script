Complete List of Supported Case Conversions

UPPERCASE

Converts all letters to uppercase. Perfect for headings, emphasis, and constants in programming.

Example: "hello world" → "HELLO WORLD"

lowercase

Converts all letters to lowercase. Ideal for URLs, email addresses, and consistent formatting.

Example: "HELLO WORLD" → "hello world"

Title Case

Capitalizes the first letter of each word. Perfect for titles, headings, and proper nouns.

Example: "hello world" → "Hello World"

Sentence case

Capitalizes the first letter of each sentence. Great for paragraph formatting and content editing.

Example: "hello world. how are you?" → "Hello world. How are you?"

camelCase

First word lowercase, subsequent words capitalized with no spaces. Essential for JavaScript and programming.

Example: "hello world" → "helloWorld"

PascalCase

All words capitalized with no spaces. Commonly used for class names in programming languages.

Example: "hello world" → "HelloWorld"

snake_case

Words separated by underscores in lowercase. Popular in Python, Ruby, and database naming.

Example: "hello world" → "hello_world"

kebab-case

Words separated by hyphens in lowercase. Commonly used for URLs, CSS classes, and file names.

Example: "hello world" → "hello-world"

CONSTANT_CASE

All uppercase with underscores. Standard for constants and environment variables in programming.

Example: "hello world" → "HELLO_WORLD"

dot.case

Words separated by dots in lowercase. Used for object properties and configuration keys.

Example: "hello world" → "hello.world"

path/case

Words separated by forward slashes. Perfect for file paths and URL structures.

Example: "hello world" → "hello/world"

aLtErNaTiNg CaSe

Alternating uppercase and lowercase letters. Fun for creative text and social media posts.

Example: "hello world" → "hElLo WoRlD"

Frequently Asked Questions

What is the difference between camelCase and PascalCase?

camelCase starts with a lowercase letter (firstName), while PascalCase starts with an uppercase letter (FirstName). Both remove spaces and capitalize subsequent words.

When should I use snake_case?

snake_case is commonly used in Python, Ruby, and database naming conventions. It's also preferred for constants in many programming languages when combined with uppercase.

Is kebab-case good for SEO?

Yes! kebab-case is excellent for URLs and file names as it's readable, SEO-friendly, and widely supported across web platforms and search engines.

Can I convert large amounts of text?

Absolutely! Our tool can handle text of any length, from single words to entire documents, with instant conversion and easy copying.
